# Javascript의 typeof
```
- Undifined : "undefined"
- Null      : "object"
- Boolean   : "boolean"
- Number    : "number"
- String    : "string"
- Symbol    : "symbol"
- Object    : "object" ( call 구현 x )
- Object    : "function" ( call 구현 o)
```
